
### Instructions for adding screenshots:

1. Run your app locally.
2. Capture screenshots of:  
   - Home page (list of posts)  
   - Single post page with comments  
   - Create/edit post form  
   - Login/Register page
3. Create a folder `client/screenshots/` and save images as:  
   - `home.png`  
   - `single-post.png`  
   - `create-post.png`  
   - `login.png`
4. Commit the `screenshots` folder and update README paths if needed.
